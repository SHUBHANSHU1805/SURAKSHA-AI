# 🚔 Trinetra AI - Crime Mapping & Predictive Policing Platform

A comprehensive web-based crime mapping and predictive policing platform designed to visualize crime data, analyze trends, and provide actionable insights for law enforcement.

## ✨ Features

- 📊 **Interactive Dashboard** - Real-time crime statistics and heatmap visualization
- 📝 **FIR Submission** - Digital form for submitting First Information Reports
- 🗺️ **Crime Heatmap** - Visual representation of crime hotspots
- 🚨 **Patrol Management** - Plan, schedule, and track patrol routes
- 📈 **Advanced Analytics** - Crime trends, distributions, and predictions
- 🌐 **Multi-language Support** - English and Hindi interface
- 🌓 **Dark/Light Mode** - Toggle between themes
- 📱 **Responsive Design** - Works on all devices

## 🛠️ Tech Stack

- **Frontend:** React 18
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Build Tool:** Create React App
